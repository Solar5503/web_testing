# Web testing
## MaX.Technology
В рамках учебной деятельности было [проведено](https://github.com/Solar5503/web_testing/tree/main/MaX.Technology) ручное функциональное тестирование ключевых модулей интернет-магазина  [MaX.Technology](https://mxt.su/) <br>
Основное внимание уделено регистрации и авторизации пользователей, поиску и фильтрации товаров, работе с корзиной и оформлением заказа, функциональности карточки товара. <br>
Тестирование проекта осуществлялось с использованием тестовой документации - чек-листов и тест-кейсов под ОС Windows 10 версии 2004 в браузерах Chrome, Firefox, Edge. Оно было произведено вручную по методу черного ящика, при этом были задействованы следующие типы тестирования: функциональное, тестирование пользовательского интерфейса, кроссбраузерное и адаптивное.<br>
Инструментами, которые использовались при тестировании проекта, были: TestRail и Excel для чек-листов и тест-кейсов, для баг-репортов – Excel и Jira, DevTools для проверки адаптивности.<br>
Тестирование кроссбраузерности (Chrome, Firefox, Edge) выполнялось по заранее подготовленным чек-листам (5 листов по основным функциям проекта). <br>
Всего было создано и запущено 76 тест-кейсов тестирования критического пути, из которых 87 % были успешно выполнены. Найдено 15 дефектов, из которых 10 — серьёзные (Major). <br>
